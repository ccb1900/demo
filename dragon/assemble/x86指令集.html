# x86指令集

# 向下兼容

兼容旧的处理器指令集。

# 变长指令

1-15 字节，多数为2-3字节长度

# 多种寻址方式（可访问不对齐内存地址）

指令集的通用寄存器个数有限 32位8个，64位16个。

至多有一个操作数在内存中，另一个操作数为立即数或者寄存器。

1 2
%rax %eax
%rbx %ebx
%rcx %ecx
%rdx %edx
%rsi %esi
%rdi %edi
%rsp %esp
%rbp %ebp
%r8 %r8d
%r9 %r9d
%r10 %r10d
%r11 %r11d
%r12 %r12d
%r13 %r13d
%r14 %r14d
%r15 %r15d

# 缺点

兼容导致指令集越来越大，越复杂。

类RISC内核，采用micro-op模式进行翻译，功耗增大，所以低功耗领域不适合。

对于很多领域，资源利用率比较低。
