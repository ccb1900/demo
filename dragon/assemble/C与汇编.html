# 变址寻址

常见形式

D(RB,RI,S) => Mem[Rem[Rb]+S*Reg[Ri]+D]

D: 常量（地址偏移量）

RB: 基址寄存器：8个通用寄存器之一

RI： 索引寄存器：%esp 不作为索引寄存器

一般 %ebp 也不用做这个用途

S：比例因子 1，2，4，8

还有一些可用的变形。
