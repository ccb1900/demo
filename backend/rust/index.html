# rust

# 常用命令

本地查看rust book

rustup docs --book

本地查看rust 版本

rustc --version

更新rustup

rustup update

查看rust 本地文档

rustup doc

编译单独的rust文件

rustc  rustfilename.rs

查看cargo 版本

cargo --version

新建项目

cargo new helloword

构建

cargo build

构建发行版

cargo build --release

检查项目是否可编译

cargo check

格式化工具 rustfmt

rust 中，变量默认是不可变的。使用let声明变量。

match 关键字匹配。

变量可重复声明，常用于类型转换。
